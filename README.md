# 🚀 Meu Portfólio Profissional

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido como projeto prático no curso de **Desenvolvimento de Sistemas do SENAI Jandira**. O objetivo é centralizar meus projetos, habilidades técnicas e trajetórias de aprendizado em uma interface responsiva e profissional.

> 🔗 **Acesse o projeto online:** [https://alvesthzin-art.github.io/meu-portfolio/](https://alvesthzin-art.github.io/meu-portfolio/)

---

### 📋 Estrutura do Projeto

O portfólio foi estruturado seguindo as melhores práticas de UX/UI e as diretrizes acadêmicas:

1.  **Header:** Identificação pessoal, boas-vindas e uma breve descrição profissional.
2.  **Habilidades (Stacks):** Divisão clara entre tecnologias de Backend (Java, Node.js, Python), Frontend (JS, HTML, CSS) e Ferramentas (Git, GitHub, UI/UX).
3.  **Projetos:** Galeria com os principais trabalhos (APIs, Landing Pages e Lógica de Programação).
4.  **Experiência e Formação:** Detalhes sobre minha jornada no SENAI e cursos extracurriculares.
5.  **Contato:** Seção dedicada com integração para E-mail, LinkedIn e WhatsApp.

---

### 🛠️ Processo de Desenvolvimento (Checklist de Entrega)

Para a gestão deste projeto, utilizei as ferramentas de governança do GitHub para garantir a entrega de todos os requisitos:

- [x] **Wiki (TAP):** Documentação do Termo de Abertura do Projeto contendo escopo e objetivos.
- [x] **Gestão de Projeto:** Criação de **Projeto Kanban** para controle de fluxo.
- [x] **Issues & Milestones:** Planejamento de tarefas com prazos definidos (Milestone: Lançamento do Portfólio).
- [x] **Desenvolvimento HTML/CSS:** Adaptação de template moderno com foco em responsividade.
- [x] **Deploy:** Hospedagem ativa via **GitHub Pages** com atualização automática.
